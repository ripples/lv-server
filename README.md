# lv-server
This repository is a submodule of the [Lecture-Viewer](https://github.com/ripples/lecture-viewer) system developed by the [ripples team at UMass Amherst](https://github.com/ripples). 

The purpose of lv-server is to act as an authentication and general purpose server for accessing and sending data from and to the [Lecture-Viewer](https://github.com/ripples/lecture-viewer) microservice stack.


### How to run
We only support running lv-server through docker and docker-compose. Follow the instructions in the [parent](https://github.com/ripples/lecture-viewer) repository readme
